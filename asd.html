Problem:
Students find it hard to use APIs and integrate them into their applications

Method:
Android Development, Twitter API, Data Analysis

Solution:
Our team discussed what tools are most appropriate for novices who are just beginning to learn about data, computing, and web architecture. To explore the beginnings of this question, we have created a prototype of the API Tutor, which is a first attempt to support students as they author interactive, data-driven web applications using various publicly available APIs such as Spotify, Marvel.

Our goal was to help novices understand:
  1. How publicly available APIs work, and
  2. How they can be leveraged in apps and websites.

Some features are:
  • Simpler template framework.
  • A "Generate UI" tool that allows data templates to be combined, linked, stylized, and downloaded to a responsive web app
  • A single JavaScript file that abstracts away some of the complexities of interacting with the various APIs
  • Better instructions for supporting high school learners
  • Explaining HTTP requests (GET / POST / PUT / PATCH / DELETE)
  • Demonstrating how authentication (OAuth) works
  • A step-by-step, getting started guide to the Spotify API
