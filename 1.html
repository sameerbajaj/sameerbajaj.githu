Shakespeare Says



Brief

To develop a web app and chrome extension that converts an inputted URL to a humourous Shakespeare quote that can be shared with others.

Problem Statement

We felt that most of the existing URL shorteners create URLs that are hard to memorize. They were also not intuitive to use and weren't quick enough to generate a URL on the go.

Solution

We developed a web application and chrome extension based on a local webserver using flask (programmed in Python). We connected a database of of 50 funny Shakespeare quotes and designed the front end in HTML, CSS, Javascript and JQuery. The chrome extension serves the purpose of speeding up the process of generating a custom URL.

Overview and Process

Google Chrome

The team wanted to create a tool none of us had ever experienced making before. We all agreed that making a Google Chrome extension would be a fun project to do, despite not knowing what it would entail. The function of our extension is simple and fun: after being downloaded to a browser, the user can click on the little Shakespeare icon and find their current browser tab URL embedded within an input field in the pop-up window.
